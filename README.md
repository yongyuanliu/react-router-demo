# react-router-demo
> 依照 React Router 官网例子进行代码实现。

## Running Locally

### Node

```
  npm run dev:build
  npm run dev:start
  npm run prod:build
  npm run prod:start
```

在本地启动项目 [http://localhost:9000/](http://localhost:9000/)
