# mat-react-template
> 前端开发模板，基于 react 技术所开发的基础性项目

## Running Locally

### Node

```
  npm run dev:build
  npm run dev:start
  npm run prod:build
  npm run prod:start
```

在本地启动项目 [http://localhost:9000/](http://localhost:9000/)

## WiKi

### entry-01
  入口文件

  * assets 公共资源文件
  * components 公共组件
  * controller 控制器/model
  * pages 页面具体逻辑
  * options 一些辅助类的文件
  * utils 工具类文件
